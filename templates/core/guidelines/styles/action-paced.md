# Template Style : Action Rythmé

## Caractéristiques générales
- Ton : Dynamique, tendu, direct
- Rythme : Rapide, haletant
- Densité : Aéré, beaucoup de blanc

## POV et narration
- Point de vue : 3ème personne limité ou 1ère personne
- Temps : Passé ou présent
- Distance narrative : Proche, immersion dans l'action

## Phrases
- Longueur moyenne : Courtes, percutantes
- Structure privilégiée : Simple, directe, fragmentée dans les moments intenses
- Ponctuation : Points. Beaucoup de points. Fragments.

## Vocabulaire
- Registre : Courant, concret
- Champs lexicaux privilégiés : Mouvement, corps, urgence
- Mots à éviter : Adverbes, qualificatifs superflus

## Descriptions
- Niveau de détail : Minimaliste, essentiel
- Sens privilégiés : Visuel et kinesthésique
- Approche : Factuelle, sensorielle, immédiate

## Dialogues
- Style : Courts, tendus, réalistes
- Attributions : Minimales
- Intégration : Entrecoupés d'action

## Introspection
- Fréquence : Rare pendant l'action
- Profondeur : Surface, réactions immédiates
- Style : Flashes, pensées brèves
