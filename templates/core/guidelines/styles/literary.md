# Template Style : Littéraire Poétique

## Caractéristiques générales
- Ton : Lyrique, évocateur, contemplatif
- Rythme : Variable, musical
- Densité : Dense, travaillé

## POV et narration
- Point de vue : Variable (souvent 3ème omniscient ou 1ère poétique)
- Temps : Variable, jeux temporels possibles
- Distance narrative : Variable, peut osciller

## Phrases
- Longueur moyenne : Variées, avec des phrases longues et sinueuses
- Structure privilégiée : Complexe, rythmée, parfois inversée
- Ponctuation : Expressive, utilisation des tirets, points de suspension

## Vocabulaire
- Registre : Soutenu, riche
- Champs lexicaux privilégiés : Nature, abstraction, métaphores
- Mots à éviter : Clichés, formulations plates

## Descriptions
- Niveau de détail : Riche, sélectif
- Sens privilégiés : Tous, synesthésies
- Approche : Métaphorique, impressionniste, symbolique

## Dialogues
- Style : Stylisés, parfois poétiques
- Attributions : Intégrées à la prose
- Intégration : Fondus dans la narration

## Introspection
- Fréquence : Fréquente
- Profondeur : Profonde, philosophique
- Style : Poétique, métaphorique, méditatif
