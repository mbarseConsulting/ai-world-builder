# Template Style : Introspectif Dense (type Robin Hobb)

## Caractéristiques générales
- Ton : Intime, réflexif, parfois mélancolique
- Rythme : Lent, contemplatif, avec des accélérations ponctuelles
- Densité : Dense, chaque phrase porte du sens

## POV et narration
- Point de vue : 1ère personne
- Temps : Passé
- Distance narrative : Très proche, le lecteur est dans la tête du personnage

## Phrases
- Longueur moyenne : Longues, avec des subordonnées
- Structure privilégiée : Complexe, sinueuse
- Ponctuation : Utilisation de tirets pour les apartés mentaux

## Vocabulaire
- Registre : Soutenu mais accessible
- Champs lexicaux privilégiés : Émotions, sensations, mémoire
- Mots à éviter : Jargon moderne, expressions trop familières

## Descriptions
- Niveau de détail : Sélectif mais profond
- Sens privilégiés : Tous, avec emphase sur le ressenti
- Approche : Les descriptions passent par le filtre émotionnel du narrateur

## Dialogues
- Style : Réaliste, parfois elliptique
- Attributions : Rares, intégrées à l'action
- Intégration : Fortement mêlés aux pensées du narrateur

## Introspection
- Fréquence : Dominante
- Profondeur : Très profonde, analyse des motivations et doutes
- Style : Analytique et émotionnel, le personnage se questionne constamment
